What is Edge Computing?

{{page.topic}} is a distributed computing paradigm approach that brings computation and data storage closer to the location where it is needed, to improve response times and save bandwidth.

think of Cloud Computing? {{page.topic}} is the extension of Cloud Computing, in that it processes the data at the edge device (example of edge devices: sensor, routers, solar panel, autonomous car, wearable device).

{{page.topic}} enables the systems and applications to be more efficient, faster and most importantly it opens up opportunities that weren't possible before.
